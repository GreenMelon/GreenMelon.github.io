JavaScript Regular Expression

1 创建正则的方式

1.1 literal

1.2 new RegExp()

2 正则的修饰符

g: global，全局匹配
i: insensitive/ignoreCase，忽略大小写
m: multiline，多行匹配，修改^和$在正则表达式中的作用，让它们分别表示行首和行尾

3 正则的基本规则

单个字符的表示

元字符/元符号 匹配情况
. 除换行符\n外的任意字符
[…] 括号中的字符集里的任意字符(warning:不管括号里有多少字符，只匹配一个字符)
[^...] 不在括号中的字符集里的任意字符(^取反)
\d 数字，相当于[0-9]
\D 非数字，相当于[^0-9]
[a-z] 小写字母
[A-Z] 大写字母
\w 字母数字下划线，相当于[a-zA-Z0-9_]
\W 非字母数字下划线，相当于[^a-zA-Z0-9_]

选择的表示

元字符/元符号 匹配情况
`item1\ item2\ itemn` item1或item2或itemn中的其中一个

匹配的次数

元字符/元符号 匹配情况
x? 0个或1个x，即{0,1}
x* 任意个x，即{0,}
x+ 至少1个x，，即{1,}
(xyz)+ 至少1个(xyz)，，即{1,n}
x{n} n个x
x{n,} n个以上的x
x{m,n} 至少m个x，至多n个x

匹配的范围

元字符/元符号 匹配情况
^ 目标字符串的开始(写在正则的开头)
$ 目标字符串的结尾(写在正则的结尾)

空白字符的表示

元字符/元符号 匹配情况
\n 换行符
\r 回车符
\t 制表符
\ 垂直制表符
\ 换页符
\s 所有不可见字符，等同于[\n\r\t\v\f]
\S 所有可见字符(即\s以外的字符)
\s\S 一切字符

锚字符的表示

元字符/元符号 匹配情况
\b 单词边界，词在[ ]内时无效
\B 非单词边界

贪婪模式与非贪婪模式

@@贪婪：默认情况，尽可能多地匹配字符
| 贪婪模式 | 匹配情况 | 非贪婪模式 | 匹配情况 |
| : | : | : | : |
| x? | 0个或1个x，即{0,1} | x?? | 0个x，即不匹配 |
| x* | 任意个x，即{0,} | x*？ | 0个x，即不匹配 |
| x+ | 至少1个x，，即{1,} | x+？ | 1个x |
| x{n} | n个x | x{n}? | n个x |
| x{n,} | n个以上的x | x{n,}? | n个x |
| x{m,n} | 至少m个x，至多n个x | x{m,n}? | m个x |

子正则表达式

避免保存子正则匹配的内容

@@方法：(?:pattern)

反向引用

@@形式：/(子正则表达式)\1/

正向预查1

@@形式：(?=pattern)
@@description: 要匹配的字符串，后面必须紧跟着pattern
@@WARNING: 括号里的pattern并不参与真正的匹配，只是检查一下后面的字符是否符合要求而已

正向预查2

@@形式：(?!pattern)
@@description: 要匹配的字符串，后面不能跟着pattern
@@WARNING: 括号里的pattern并不参与真正的匹配，只是检查一下后面的字符是否符合要求而已

4 正则的方法

test: 验证
@format: regexp.test(string); // return Boolean
match: 捕获
@format: string.match(regexp); // return Array
replace: 替换
@format: string.replace(regexp,’new string’); // return String
exec: 捕获
@format: regexp.exec(string); // return Array
